## Getting Started

1. Clone the repository:

   ```
   git clone https://github.com/AagamAJ/My-Resume.git
   ```
2. Move to the Folder:

   ```
   cd MyResume
   ```

3. Open the `index.html` file in a web browser to access My Resume.
